About the Project

A deep learning system for binary classification of kidney MRI scans — distinguishing tumor from normal tissue.

Overview

What it does

kidneyScan AI takes a kidney MRI image as input and outputs a prediction — either Tumor or Normal — along with a confidence score derived from the model's raw sigmoid output.

The model is a convolutional neural network trained on labelled MRI datasets. Images are resized to 224×224 pixels, normalized, and passed through the network. The output neuron uses a sigmoid activation; values above 0.5 are classified as Tumor.
